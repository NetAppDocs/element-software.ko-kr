= 로그 서버
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
loggingServer 객체는 스토리지 클러스터에 대해 구성된 로깅 호스트에 대한 정보를 저장합니다.  사용 `GetRemoteLoggingHosts` 현재 로깅 호스트를 보고 다음을 사용합니다. `SetRemoteLoggingHosts` 현재 및 새로운 로깅 호스트의 원하는 목록을 설정합니다.



== 개체 멤버

이 개체에는 다음 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
호스트
 a| 
로그 서버의 FQDN 또는 IP 주소
 a| 
문자열



 a| 
포트
 a| 
로그 서버와 통신하는 데 사용되는 포트 번호
 a| 
정수

|===
