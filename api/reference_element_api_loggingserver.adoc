= 로그 서버
:icons: font
:imagesdir: ../media/


[role="lead"]
loggingServer 객체에는 스토리지 클러스터에 구성된 로깅 호스트에 대한 정보가 포함되어 있습니다. GetRemoteLoggingHosts 를 사용하여 현재 로깅 호스트가 무엇인지 확인한 다음 SetRemoteLoggingHosts 를 사용하여 현재 로깅 호스트와 새 로깅 호스트의 원하는 목록을 설정할 수 있습니다.

 a|
