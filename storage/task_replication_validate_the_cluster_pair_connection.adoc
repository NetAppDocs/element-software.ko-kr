= 클러스터 쌍 연결을 확인합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
클러스터 페어링이 완료된 후 클러스터 쌍 연결을 확인하여 복제가 성공했는지 확인할 수 있습니다.

. 로컬 클러스터에서 * 데이터 보호 * > * 클러스터 쌍 * 을 선택합니다.
. Cluster Pairs * 창에서 클러스터 페어가 연결되어 있는지 확인합니다.
. * 선택 사항: * 로컬 클러스터와 * 클러스터 쌍 * 창으로 다시 이동하여 클러스터 쌍이 연결되었는지 확인합니다.
