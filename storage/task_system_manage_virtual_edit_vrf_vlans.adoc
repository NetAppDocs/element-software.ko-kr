= VRF VLAN을 편집합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
VLAN 이름, 넷마스크, 게이트웨이 및 IP 주소 블록과 같은 VRF VLAN 속성을 변경할 수 있습니다.

. Cluster * > * Network * 를 클릭합니다.
. 편집할 VLAN의 작업 아이콘을 클릭합니다.
. 편집 * 을 클릭합니다.
. VLAN 편집 * 대화 상자에 VRF VLAN에 대한 새 속성을 입력합니다.
. 변경 내용 저장 * 을 클릭합니다.
