= 볼륨 쌍을 삭제합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
두 볼륨 간의 쌍 연결을 제거하려면 볼륨 쌍을 삭제할 수 있습니다.

. 데이터 보호 * > * 볼륨 쌍 * 을 선택합니다.
. 삭제할 볼륨 쌍의 작업 아이콘을 클릭합니다.
. 삭제 * 를 클릭합니다.
. 메시지를 확인합니다.
