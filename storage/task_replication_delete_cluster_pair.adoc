= 클러스터 쌍을 삭제합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
쌍에 있는 클러스터 중 하나의 Element UI에서 클러스터 쌍을 삭제할 수 있습니다.

. 데이터 보호 * > * 클러스터 쌍 * 을 클릭합니다.
. 클러스터 쌍의 작업 아이콘을 클릭합니다.
. 결과 메뉴에서 * 삭제 * 를 클릭합니다.
. 작업을 확인합니다.
. 클러스터 페어링의 두 번째 클러스터에서 단계를 다시 수행합니다.
