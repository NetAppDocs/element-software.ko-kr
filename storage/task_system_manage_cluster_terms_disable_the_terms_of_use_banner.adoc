= 사용 약관 배너를 사용하지 않도록 설정합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
이용 약관 배너를 사용하지 않도록 설정할 수 있습니다. 배너가 비활성화된 경우 사용자는 더 이상 Element UI를 사용할 때 사용 약관에 동의하도록 요청되지 않습니다.

* 사용 약관을 구성하려면 클러스터 관리자 권한이 있어야 합니다.
* 사용 약관이 활성화되어 있는지 확인합니다.
+
.. 사용자 * > * 이용 약관 * 을 클릭합니다.
.. 비활성화 * 를 클릭합니다.
