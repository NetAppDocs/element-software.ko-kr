= 다중 요소 인증을 사용합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
MFA(Multi-factor Authentication)는 SAML(Security Assertion Markup Language)을 통해 타사 ID 공급자(IdP)를 사용하여 사용자 세션을 관리합니다. 관리자는 MFA를 사용하여 암호 및 텍스트 메시지, 암호 및 전자 메일 메시지 등 필요에 따라 추가 인증 요소를 구성할 수 있습니다.
