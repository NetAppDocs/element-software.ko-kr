= 볼륨 작업
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SolidFire 시스템은 볼륨을 사용하여 스토리지를 프로비저닝합니다. 볼륨은 iSCSI 또는 파이버 채널 클라이언트가 네트워크를 통해 액세스하는 블록 디바이스입니다. 관리 탭의 볼륨 페이지에서 노드에서 볼륨을 생성, 수정, 클론 복제 및 삭제할 수 있습니다. 볼륨 대역폭 및 I/O 사용량에 대한 통계도 볼 수 있습니다.
